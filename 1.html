HTML

Hypertext Markup Language (HTML) is the standerd markup language for creating web pages and web applications. Web browsers recieve HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes th structure of a web page semantically and originally included cures for the appearance of th document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provies a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links quotes and other items. HTML elemets are delineated by tags, writtend using angle brackets.
